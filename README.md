# BRIDGE Scoping Review – Epidemic Governance

Timeline: September 2025 – Present

## Overview
Multi-institutional scoping review examining the role of regional organizations in pandemic prevention, preparedness, response, and resilience (PPPR).

## Key Contributions
- Screened 2,500+ abstracts; currently conducting full-text review
- Systematic data extraction and coding using Covidence
- Organized findings into thematic categories to support synthesis and reporting

## Methods & Tools
PRISMA framework, systematic review methodology, Covidence, Excel

## Impact
Findings inform research outputs and policy guidance for strengthening epidemic governance and public health preparedness.
